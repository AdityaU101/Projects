## 👋 About This Project

Poetry Haven is a frontend UI mockup for a poetry platform where users can explore poems, discover poets, and browse genres. The project demonstrates component-based React design, custom CSS theming, and React Bootstrap for layout, with a clear structure ready to be extended with React Router and a backend API.

---

## ✨ Features

- **Hero section** with a full-width banner, poet search bar, and subscribe CTA
- **Featured Poets** section with curated poet spotlight cards
- **Genre Blocks** for browseable poetry categories
- **Header and Footer** with persistent navigation and Font Awesome brand icons
- Component-scoped CSS for clean style isolation

---

## 🛠️ Tech Stack

`React 18` `React Bootstrap` `Bootstrap 5` `Font Awesome` `CSS3`

## 🚀 Getting Started

```bash
cd client
npm install
npm start
# App runs on http://localhost:3000
```
