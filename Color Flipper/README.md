## 👋 About This Project

Color Flipper is a simple vanilla JavaScript project with two modes:

- **Simple mode** cycles through a curated list of named/rgba colors
- **Hex mode** generates a fully random 6-character hex color code on every click

Both modes update the page background live and display the current color value, making it a handy micro-tool for quick color exploration.

---

## ✨ Features

- One-click random color generation
- Two modes: named colors (`index.html`) and full hex range (`hex.html`)
- Live color code display in the UI
- Zero dependencies: pure HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack

`HTML5` `CSS3` `Vanilla JavaScript`

---

## 🚀 Getting Started

No build step needed. Just open either file directly in your browser:

```bash
open index.html    # Simple (named colors) mode
open hex.html      # Hex mode
```
