## 👋 About This Project

Counter App is a beginner-friendly vanilla JavaScript project demonstrating DOM manipulation and basic state management. Users can increment or decrement a live counter and snapshot the current value into a running history list, all without a page reload.

---

## ✨ Features

- Real-time counter display
- Increment and decrement controls
- **Save** button that appends the current value to a persistent on-page history
- Zero dependencies: pure HTML, CSS, and JavaScript

---

`HTML5` `CSS3` `Vanilla JavaScript`
